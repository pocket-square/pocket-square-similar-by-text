= HOWTO: start service as docker container

== Cleaning previously started container

[source,shell]
----
docker ps -a
docker kill pocket_square_similar_by_text; docker rm pocket_square_similar_by_text
----

== Start service

[source,shell]
----
docker pull pocketsquare/pocket-square-similar-by-text:$TAG
docker run -dit --name pocket_square_similar_by_text --link pocket_square_articles -p 28105:5000 pocketsquare/pocket-square-similar-by-text:$TAG
----
